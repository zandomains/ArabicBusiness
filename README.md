# ArabicBusiness (static site)

Static HTML/CSS/JS website.

## Structure
- `index.html` and other `*.html`: site pages
- `assets/css/styles.css`: global styles
- `assets/js/script.js`: global scripts
- `assets/img/`: images (SVG/PNG, etc.)
- `unused/`: assets kept for reference but not used by pages

## Run locally
Open `index.html` in your browser.
